# ☕ Bean & Byte

**Bean & Byte** is a modern café + restaurant mobile application designed to deliver a seamless food ordering experience with a clean, minimal interface and powerful backend integrations.

---

## 🚀 Features

### 🛒 Smart Ordering Experience

* Browse menu with categories and rich UI
* Add/remove items with real-time updates
* Dynamic cart management

### 💳 Secure Payments

* Integrated online payment system
* Smooth checkout experience
* Order confirmation and tracking

### 📍 Location-Based Services

* Pickup and delivery support
* Address selection and management
* Real-time order status

### 👤 User Management

* Authentication (Login/Signup using Firebase)
* Profile management with image upload
* Persistent user sessions

### 🧾 Order Management

* Order history tracking
* Live order updates
* Cancel & manage orders easily

### 🎨 Minimal UI/UX

* Clean, modern design
* Smooth animations
* Optimized for performance

---

## 🛠️ Tech Stack

### Frontend

* **Flutter** — Cross-platform UI development
* **Dart** — Programming language

### Backend & Services

* **Supabase**

  * PostgreSQL Database
  * Storage (User images & assets)

### Authentication

* **Firebase Authentication**

  * Email & Password login
  * Secure user session management

### Payments

* **Razorpay** — Secure payment gateway integration

---

## 📦 Architecture

* Modular and scalable code structure
* State management using **Provider**
* API-driven dynamic UI updates
* Clean separation of UI, business logic, and services

---

## ⚡ Key Highlights

* Real-time UI updates without full screen refresh
* Efficient state handling for cart & checkout
* Clean and maintainable codebase
* Designed for scalability (future delivery system integration possible)
